## Affected Pages/Routes
- `scripts/capture_refactor_regression.py` — capture baseline/candidate rows, emit unique artifact dirs, and surface local-only / missing-prerequisite failures honestly.
- `scripts/analyze_refactor_regression.py` — pair rows on stable coordinates, enforce suite-specific thresholds, and fail fast on missing rows or breached policies.
- `docs/refactor-regression.md` — operator workflow for baseline capture, candidate capture, comparison, and local-only guidance.
- `.planning/phases/04-refactor-regression-harnesses-for-vlm-territory-coverage-and/04-LOCAL-PROBE-RESULTS.md` — local proof record with exact commands, coordinates, artifact paths, and analyzer outcomes.

## Key Interactions to Verify
- Capture the same coordinate tuple twice and confirm `results.jsonl` appends while each run lands in a distinct `artifact_dir`.
- Run a direct-VLM suite in an unprovisioned environment and confirm the harness fails loudly into an actionable error row instead of pretending the path is cloud-safe.
- Run an OpenClaw push-model suite and confirm `visited_cells` is derived from replay artifacts when the runner does not return it directly.
- Attempt a local-only suite without `--allow-local` and confirm the CLI blocks with a cloud/local-split message.
- Compare baseline vs candidate rows with matching coordinates and confirm missing-row, threshold-pass, and threshold-fail paths all behave deterministically.

## Edge Cases
- `variant` absent for a suite: normalized row must still include a nullable `variant` key.
- Baseline/candidate pair uses the same commit for first-workflow proof: analyzer should pass and the write-up must say it was a workflow-proof pair.
- Repeated captures of the same coordinates on the same label: earlier artifacts must remain intact.
- Missing `.env` sourcing on local preflight: operator path must call out `set -a && source .env && set +a` before provider checks.
- `openclaw-demo` extractor sees replay artifacts but no direct `visited_cells` return field.
- `openclaw-autonomous` extractor sees `run_result.json` without `summary.json` yet, or vice versa.

## Critical Paths
- Plan 01 contract wall: replay summary keys, prompt-image ordering, row-schema contract, and trace-schema additive/exact rules.
- Plan 02 capture path: registry lookup -> coordinate normalization -> unique artifact dir -> append-only JSONL row -> actionable error-row fallback.
- Plan 03 OpenClaw path: local-only guard -> replay/run_result extraction -> stable coordinates -> structured row with no secrets.
- Plan 04 compare path: pair rows on stable coordinates -> enforce suite thresholds -> emit `summary.md`/`summary.json` -> non-zero exit on breach.
