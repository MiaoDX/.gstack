## What to verify

Phase 2 ships `examples/openclaw_demo.py` — a standalone, multi-agent AI2-THOR demo
routed through a local OpenClaw Gateway container. CI publishes its replay GIF +
HTML report to GitHub Pages. Tests must prove: the transport contract works, the
demo produces artifacts, and the published URLs return 200.

## Key interactions to verify

- **CLI:** `python examples/openclaw_demo.py --agents 2 --steps 20 --output-dir output/openclaw/demo` — runs to completion, exits 0, writes `replay.gif` + `report.html`
- **Gateway readiness:** `curl -H "Authorization: Bearer $TOKEN" http://localhost:18789/readyz` → HTTP 200 within 60s
- **Bridge HTTP:** `OpenClawBridge.step(...)` POSTs to `/tools/invoke` with `sessionKey`, gets back `{reasoning, action}` — existing test suite (27 tests in tests/test_bridge.py)
- **Report interactivity:** step slider advances, per-agent FPV switches frame, VLM reasoning pane updates

## Edge cases

- **Stale work_dir:** `.openclaw-tmp/` has files owned by container UID from prior run → host user write fails → A2 fix: `shutil.rmtree(work_dir, ignore_errors=True)` on demo start
- **Session collision:** same session_prefix across two local runs → A1 fix: `session_prefix=f"roboclaws-demo-{int(time.time())}"`
- **macOS File Sharing:** `$PWD` outside Docker Desktop's shared folders → Gateway can't read frame paths → A4 fix: troubleshooting note in docs/openclaw-local.md
- **Image drift:** upstream rebuilds `ghcr.io/openclaw/openclaw:latest` → A3 fix: pin to `ghcr.io/openclaw/openclaw:2026.4.14`
- **Empty/zero paths:** `_write_image(None, ...)` writes 1×1 JPEG placeholder (existing behavior, covered by test_provider_handles_missing_images)
- **Gateway down:** `OpenClawUnavailable` raised with clear message (existing tests: test_step_404, test_step_401, test_step_500, test_step_connection_error, test_step_timeout)

## Critical paths

- **End-to-end demo:** `docker run` Gateway → `python examples/openclaw_demo.py` → frames written → Gateway invokes skill → action returned → engine steps → repeat 20× → GIF + report written
- **CI publish flow:** `openclaw-smoke` CI job green → artifact uploaded → `publish-pages` downloads → site assembled → GitHub Pages deploys → URLs return 200
- **Cross-platform:** demo runs on Linux (CI) + macOS Docker Desktop (developer laptop) + Windows WSL2 (not supported in docs, but shouldn't crash)
